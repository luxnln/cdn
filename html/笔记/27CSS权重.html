CSS权重

CSS权重指的是样式的优先级，有两条或多条样式作用于一个元素，权重高的那条对样式元素起作用，权重相同的，后写的样式会覆盖前面写的样式

权重的等级

可以把样式的应用方式分为几个等级，按照等级来计算权重

  1. limportant，加在样式属性值后，权重值为10000
  2. 内联样式，如：sytle=""，权重值为1000
  3. ID选择器，如：#content，权重值为100
  4. 类，伪类和属性选择器，如：countent、hover权重值为10
  5. 标签选择器和伪元素选择器，如：div、p、before权重为1
  6. 通用选择器（*）、自选择器（>）、相邻选择器（+）、同胞选择器（-）权重为0
权重计算实例
  1. 实例一
    <style type="text/css">
        div{
           color:red !import;
        } </style>
        ...
        <div style="color:blue">这是一个div元素</div>
        <!-- 两条样式同时作用一个div，上面的样式权重值为10000+1，下面那的行间样式的权重值为1000，所以文字的最终颜色为red -->
  2. 实例二
    <style type="text/css">
        #content div.main_content h2{
           color:red;
        }
        </style>
        #content .main_content h2{
           color:blue;
        }
        ...
        <div id="content">
           <div id="content">
              <h2>这是一个h2标题</h2>
           </div>
        </div>
        <!--
        第一条样式的权重的计算：100+1+10+1，结果为112；
        第二条样式的权重的计算：100+10+1，结果为111；
        h2标题的最终颜色的为red
        -->
