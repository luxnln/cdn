函数

函数就是重复执行的代码片

函数定义与执行
<script type="text/javascript">
    //函数定义
    function fnAlert(){
       alert('hello!');
    }
    //函数执行
    fnAlert();
</script>
变量与函数预解析
JavaScript解析过程分为两个阶段，先是编译阶段，然后执行阶段在编译阶段会将function定义的函数提前，并且将var定义的变量提前，将它赋值为undefined。
    <script type="text/javascript">
       //与解析会让函数的声明和定义提前
       fnAlert();   //弹出hello!

       //函数的预解析会让函数的声明提前
       alert(iNum);   //弹出undifined
       alert(iNum1); //使用一个未声明的变量，出错！
       var iNum = 12;

       function fnAlert(){
          alert('hello!');
       }
    </script>

函数'return'关键字
函数中'return'关键字的作用：
  1. 返回函数执行的结果
  2. 结束函数的运行
  3. 阻止默认行为
<script type="text/javascript">
       function fnAdd(iNum1, iNum2){
          var iRs = iNum1 + iNum2;
          return iRs;
          alert('here!');
       }

       fnAlert(12345);
</script>
